# Gym Management System

A full-stack Gym Management Platform built with Flutter, React, Node.js, Express.js, MongoDB, JWT Authentication, and Razorpay.

## Live Demo
🌐 Website: https://your-vercel-link.vercel.app

## Features

- User Authentication & Authorization
- Membership Management
- Attendance Tracking
- Plan Management
- Payment Integration (Razorpay)
- Dashboard Analytics
- Member Portal
- Admin Portal

## Tech Stack

### Frontend
- React.js
- TypeScript
- Tailwind CSS

### Mobile Apps
- Flutter

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
